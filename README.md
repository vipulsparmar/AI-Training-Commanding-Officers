# Military AI Training for Commanding Officers

This repository contains a suite of 5 realistic operational exercises designed to train senior staff officers and commanding officers on the practical application of AI in staff work, intelligence analysis, and decision support.

## 📁 Repository Structure

### [MASTER_PROMPT.md](./MASTER_PROMPT.md)
The core logic used to generate high-fidelity military training curriculum. This document acts as the "Engine" for creating new exercises.

### 🎖️ Exercises
Each exercise is organized into a dedicated folder with separate data and instructional components:

1.  **[Exercise 01: Intelligence Digest](./Exercise_01_Intelligence_Digest/)**
    *   *Focus:* SIGINT/HUMINT synthesis and pattern recognition.
2.  **[Exercise 02: SITREP Generator](./Exercise_02_SITREP_Generator/)**
    *   *Focus:* Transforming chaotic field notes into structured reporting.
3.  **[Exercise 03: Risk Identification](./Exercise_03_Risk_Identification/)**
    *   *Focus:* Using AI as a "Red Team" to expose operational blind spots.
4.  **[Exercise 04: COA Generator](./Exercise_04_COA_Generator/)**
    *   *Focus:* Brainstorming tactical courses of action with speed and lateral thinking.
5.  **[Exercise 05: Decision Briefing](./Exercise_05_Decision_Briefing/)**
    *   *Focus:* Distilling complex operations into high-level command briefs.

## 🛠️ How to Use
1.  **Navigate** to a specific exercise folder.
2.  **Read** the `data_readme.md` to understand the operational context.
3.  **Follow** the instructions in `exercise_readme.md`.
4.  **Use** the provided AI Prompts in your LLM of choice (e.g., Gemini, ChatGPT, Claude) to complete the tasks.
